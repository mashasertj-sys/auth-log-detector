#Auth Log Anomaly Detector

Инструмент для обнаружения в логах аутентификации, написанный на Python.

Запусти проект одной командой с помощью Docker:

```bash
# 1. Клонировать репозиторий
git clone https://github.com/ТВОЙ_НИК/auth-log-detector.git
cd auth-log-detector

# 2. Запустить всё одной командой
docker-compose up

# 3. Открыть в браузере
#  http://localhost:8080
